# AI Help Desk (Starter)

This is a starter template for an AI-augmented help desk system.

## Features
- Ticket CRUD API
- Knowledge Base (KB) ingestion from files
- AI “resolution suggestion” endpoint (mock provider by default)
- Simple API-key authentication
- Clean, extensible architecture suitable for production expansion
